Main File Structure

All the file are well organized, its so easy to work with the template.
1. Unzip the files.
2. Open "robofume" folder.
You will find the 404.html, about.html, blog.html, cart.html, checkout.html, contact.html, index.html, login.html, partner.html, pricing.html, privacy.html, product.html, register.html, service.html, single-blog.html, single-product.html, single-service.html, team.html file on the root that will be edited.
There are 1 directory called "assets".
Also you will see 4 directories under assets directory "css, js, fonts, img, flaticon".
In the css folder Content you will find style.css and other styling files the all the other files expect the htmls.
Js folder contain only the jquery library. Fonts folder contain essetial font files for fonts.
